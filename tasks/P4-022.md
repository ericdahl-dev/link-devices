## Goal

Support meters other than 4/4 — odd time signatures, with a quarter- or
eighth-note beat unit — so the bar-1 accent, the visual metronome, and any
bar-quantized features land on the real downbeat.

## Inspiration

JMK CLOCKstep:MULTI "Time Signatures" — odd meters with quarter or eighth beat
lengths; the top of each bar drives the click accent and quantized features
(https://jmkmusicpedals.com/products/clockstep-multi).

## Current state

`METRO_QUANTUM` is hardcoded to `4.0` in `ks_main.c` (Link carries no meter, so
we assume 4/4). This is used by the metronome accent AND the LED bar-position
chase (`metro_strip`), so both assume 4 beats/bar.

## Scope

- Config: `beats_per_bar` (e.g. 2..16) + beat unit (quarter / eighth). Pure,
  validated, on the /live path.
- Thread it through: metronome accent (bar-1), `metro_strip` quantum (chase/fill
  wrap), and any future bar-quantize (P4-025).
- Web UI: a meter control.

## Deliverable

Set 3/4, 7/8, etc.; the accent + LED downbeat land on beat 1 of the bar. Pure
logic host-tested (metronome + metro_strip already take `quantum` as a param, so
this is mostly plumbing a config value instead of the constant).
