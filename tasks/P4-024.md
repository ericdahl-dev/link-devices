## Goal

Save and recall whole-device configurations (tempo behaviour, outputs, metronome,
LED, meter, ...) so you can switch between songs/setups without re-entering
settings.

## Inspiration

JMK CLOCKstep:MULTI "Presets" — save/recall up to 40
(https://jmkmusicpedals.com/products/clockstep-multi).

## Current state

One live `KsConfig` in NVS.

## Scope

- Store N preset slots in NVS (keyed blobs) — build on P4-014's versioned config
  codec so a firmware struct change doesn't corrupt saved presets.
- Web UI: save-to-slot / recall-from-slot / name a slot.
- Optional later: recall a preset via MIDI Program Change (pairs with a MIDI-in
  remote-control feature).

## Deliverable

Save the current config to a slot and recall it; presets survive reboot and (via
P4-014) firmware updates. Depends on P4-014 landing first for safe storage.
