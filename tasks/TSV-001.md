## Goal

Create the `esp32/X32ToastSaver/` Arduino sketch directory for the hardware Toast Saver device. This is a standalone ESP32-S3 firmware that:
- Captures audio from an INMP441 I2S MEMS microphone
- Detects feedback acoustically via FFT
- Sends OSC TEQ notch commands to an XR18 over WiFi

Target board: **Waveshare ESP32-S3-Touch-LCD-4.3** (800×480, capacitive touch, ESP32-S3 with 8MB PSRAM).

- [ ] `X32ToastSaver/X32ToastSaver.ino` exists. `setup()` connects to WiFi using `g_config.wifi_ssid/pass`, sends `/xinfo` to `g_config.mixer_ip:10024`, and prints the reply to Serial. `loop()` is empty stub.
- [ ] `X32ToastSaver/build_opt.h` contains:
  ```
  -DBOARD_HAS_PSRAM
  -DARDUINO_USB_CDC_ON_BOOT=1
  ```
- [ ] `X32ToastSaver/config.h` contains TSV-specific constants (see below).
- [ ] All symlinks listed above are created and resolve correctly.
- [ ] Sketch compiles in Arduino IDE with board = **ESP32S3 Dev Module**, PSRAM = **OPI PSRAM**, USB CDC on Boot = **Enabled**.
- [ ] Serial output on first boot: `[TSV] connecting to <ssid>...`, `[TSV] connected`, `[TSV] XR18 handshake OK`.

## config.h contents

```c
#pragma once

// Detection
#define TSV_SAMPLE_RATE_HZ    48000
#define TSV_FFT_SIZE          1024          // points; frequency resolution = 48000/1024 ≈ 46.9 Hz/bin
#define TSV_THRESHOLD_DB      20.0f         // dB above noise floor to trigger
#define TSV_CUT_DB            -6.0f         // notch depth
#define TSV_RELEASE_SEC       10.0f         // seconds before notch is released
#define TSV_BASELINE_FRAMES   40            // frames to establish noise floor (~2s at ~20 fps)
#define TSV_RATE_THRESHOLD_DB 15.0f         // dB/frame rate-of-change to confirm feedback onset

// I2S mic (INMP441) — adjust pins to match board breakout
#define TSV_I2S_BCLK          GPIO_NUM_4
#define TSV_I2S_LRCLK         GPIO_NUM_5
#define TSV_I2S_DIN           GPIO_NUM_6
#define TSV_I2S_PORT          I2S_NUM_0

// Mixer
#define TSV_MIXER_PORT        10024
#define TSV_FX_SLOT           4
#define TSV_XREMOTE_INTERVAL  9000          // ms between /xremote keepalives
```

## Notes

- Symlinks follow the same pattern as `X32MidiClock/`. Use `ln -s ../X32Link/<file> <file>` from inside `X32ToastSaver/`.
- `osc_sender.cpp` from X32Link sends BPM — that function will be replaced in TSV-004 with a notch sender. For now it can stay as-is; the file is symlinked but `osc_send_bpm()` is simply not called.
- The `/xinfo` handshake in `setup()` should retry up to 10 times with 500ms delay before giving up and restarting. Use `ESP.restart()` if no reply after retries.
- AppConfig already has `mixer_ip`, `fx_slot`, `wifi_ssid`, `wifi_pass` — no AppConfig changes needed for this task.
