## Goal

Implement `esp32/X32ToastSaver/fft_detector.cpp` and `fft_detector.h`: take a 1024-sample int32_t frame from `mic_capture_read()`, compute the power spectrum, maintain a per-bin noise floor baseline, detect feedback via amplitude threshold + rate-of-change, and return the peak GEQ par number (1–31) and its frequency.

## Why rate-of-change matters

Feedback is distinguished from loud speech by its onset speed. Speech has gradual amplitude variation; feedback builds 15–30 dB in under 100ms. Checking both `current > baseline + threshold` AND `current - prev > rate_threshold` catches feedback and ignores a guest banging the podium.

## Public interface (fft_detector.h)

```c
#pragma once
#include <stdbool.h>

typedef struct {
    int   geq_par;       /* 1–31; GEQ par number of the peak bin */
    int   fft_bin;       /* raw FFT bin index */
    float freq_hz;       /* approximate center frequency of the bin */
    float excess_db;     /* how many dB above baseline the peak bin is */
} FeedbackEvent;

/* Initialize detector state. Call once from setup() after mic_capture_begin(). */
void fft_detector_begin(void);

/* Process one frame of TSV_FFT_SIZE int32_t samples.
 * Returns true and fills *evt if a feedback event is detected; false otherwise.
 * Must be called each loop iteration immediately after mic_capture_read(). */
bool fft_detector_process(const int32_t *frame, FeedbackEvent *evt);

/* Reset baseline (call when entering a quiet period). */
void fft_detector_reset_baseline(void);
```

## Implementation guide (fft_detector.cpp)

### 1. Includes and constants

```cpp
#include "fft_detector.h"
#include "config.h"
#include <esp_dsp.h>
#include <math.h>
#include <string.h>

#define N  TSV_FFT_SIZE        /* 1024 */
#define BIN_HZ (TSV_SAMPLE_RATE_HZ / (float)N)   /* ~46.9 Hz per bin */
```

### 2. GEQ bin table (same calibrated table as the C tool)

Map the 31 GEQ 1/3-octave center frequencies to their nearest FFT bin at 48kHz/1024:

```cpp
static const float GEQ_FREQ_HZ[31] = {
    20, 25, 31.5f, 40, 50, 63, 80, 100, 125, 160,
    200, 250, 315, 400, 500, 630, 800, 1000, 1250, 1600,
    2000, 2500, 3150, 4000, 5000, 6300, 8000, 10000, 12500, 16000, 20000
};

/* Pre-computed at begin() time. GEQ_FFT_BIN[i] = round(GEQ_FREQ_HZ[i] / BIN_HZ). */
static int GEQ_FFT_BIN[31];

static int freq_to_fft_bin(float hz) {
    int b = (int)(hz / BIN_HZ + 0.5f);
    if (b < 0) b = 0;
    if (b >= N/2) b = N/2 - 1;
    return b;
}

static int fft_bin_to_geq_par(int bin) {
    int best = 0, best_dist = 99999;
    for (int i = 0; i < 31; i++) {
        int d = abs(GEQ_FFT_BIN[i] - bin);
        if (d < best_dist) { best_dist = d; best = i; }
    }
    return best + 1;
}
```

### 3. State

```cpp
static __attribute__((aligned(16))) float s_fft_buf[N * 2]; /* complex interleaved */
static float s_window[N];
static float s_power[N/2];        /* power spectrum, dB */
static float s_baseline[N/2];     /* per-bin noise floor, dB */
static float s_prev_power[N/2];   /* previous frame for rate-of-change */
static int   s_frame_count;
static bool  s_baseline_ready;
```

### 4. `fft_detector_begin()`

```cpp
void fft_detector_begin(void) {
    /* Pre-compute GEQ FFT bin positions */
    for (int i = 0; i < 31; i++)
        GEQ_FFT_BIN[i] = freq_to_fft_bin(GEQ_FREQ_HZ[i]);

    /* Hann window coefficients */
    dsps_wind_hann_f32(s_window, N);

    /* Initialize FFT tables */
    dsps_fft2r_init_fc32(NULL, N);

    memset(s_baseline, 0, sizeof(s_baseline));
    memset(s_prev_power, 0, sizeof(s_prev_power));
    s_frame_count   = 0;
    s_baseline_ready = false;
}
```

### 5. `fft_detector_process()`

```cpp
bool fft_detector_process(const int32_t *frame, FeedbackEvent *evt) {
    /* Convert int32_t → float, apply Hann window, interleave as complex */
    for (int i = 0; i < N; i++) {
        float s = (float)(frame[i] >> 8) / (float)0x7FFFFF;  /* normalize to [-1,1] */
        s_fft_buf[2*i]   = s * s_window[i];   /* real */
        s_fft_buf[2*i+1] = 0.0f;              /* imag */
    }

    /* In-place radix-2 FFT (bit-reversed output) */
    dsps_fft2r_fc32(s_fft_buf, N);
    dsps_bit_rev_fc32(s_fft_buf, N);

    /* Power spectrum (dBFS) — only first N/2 bins */
    for (int i = 0; i < N/2; i++) {
        float re = s_fft_buf[2*i];
        float im = s_fft_buf[2*i+1];
        float mag = sqrtf(re*re + im*im) / (N/2);
        s_power[i] = (mag > 1e-10f) ? 20.0f * log10f(mag) : -120.0f;
    }

    /* Baseline calibration */
    if (!s_baseline_ready) {
        if (s_frame_count == 0)
            memcpy(s_baseline, s_power, sizeof(s_baseline));
        else
            for (int i = 0; i < N/2; i++)
                s_baseline[i] = 0.9f * s_baseline[i] + 0.1f * s_power[i];
        s_frame_count++;
        if (s_frame_count >= TSV_BASELINE_FRAMES) {
            s_baseline_ready = true;
            Serial.println("[TSV] fft baseline ready");
        }
        memcpy(s_prev_power, s_power, sizeof(s_power));
        return false;
    }

    /* Exponential baseline update (slow follow) */
    for (int i = 0; i < N/2; i++)
        s_baseline[i] = 0.995f * s_baseline[i] + 0.005f * s_power[i];

    /* Detection: threshold + rate-of-change */
    int   peak_bin = -1;
    float peak_val = -999.0f;
    for (int i = 1; i < N/2; i++) {   /* skip bin 0 (DC) */
        float excess   = s_power[i] - s_baseline[i];
        float rate     = s_power[i] - s_prev_power[i];
        if (excess >= TSV_THRESHOLD_DB && rate >= TSV_RATE_THRESHOLD_DB) {
            if (s_power[i] > peak_val) { peak_val = s_power[i]; peak_bin = i; }
        }
    }

    memcpy(s_prev_power, s_power, sizeof(s_power));

    if (peak_bin < 0) return false;

    evt->fft_bin   = peak_bin;
    evt->freq_hz   = peak_bin * BIN_HZ;
    evt->geq_par   = fft_bin_to_geq_par(peak_bin);
    evt->excess_db = peak_val - s_baseline[peak_bin];
    return true;
}
```

## Acceptance Criteria

- [ ] `fft_detector.h` defines `FeedbackEvent` and the three-function interface above.
- [ ] `fft_detector.cpp` implements all three using `<esp_dsp.h>`.
- [ ] `fft_detector_begin()` called from `setup()` after `mic_capture_begin()`.
- [ ] `loop()` calls `mic_capture_read()` then `fft_detector_process()`. When a feedback event fires, prints: `[TSV] FEEDBACK bin=%d freq=%.0fHz par=%d excess=%.1fdB`.
- [ ] Sketch compiles without warnings. Must add `esp-dsp` to Arduino library manager or `platformio.ini` dependencies.
- [ ] When whistling near the mic, a feedback event fires within ~2 seconds of baseline completion.

## Notes

- `__attribute__((aligned(16)))` on `s_fft_buf` is required by `dsps_fft2r_fc32` for SIMD alignment.
- `dsps_fft2r_init_fc32(NULL, N)` uses internal tables; passing NULL avoids allocating a user buffer.
- The rate-of-change check (`rate >= TSV_RATE_THRESHOLD_DB`) adds latency of exactly 1 frame (~50ms) vs. threshold-only detection. This is intentional — it eliminates false positives from impulse transients.
- Skip FFT bins 0 (DC offset) and 1 (<47 Hz) — sub-bass feedback is rare on vocal mics and the INMP441 rolls off below ~60 Hz anyway.
- Only detect feedback in bins above `freq_to_fft_bin(80.0f)` to avoid low-frequency noise false triggers. Add a `min_detect_bin` constant to `config.h` if desired.
