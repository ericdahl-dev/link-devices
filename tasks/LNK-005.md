## Goal

Open a UDP socket to the XR18 and write the current BPM as a normalised float
to `/fx/{slot}/par/02` whenever `g_bpm_changed` is set. This is the final
output stage — Link tempo → XR18 delay time.

## Acceptance Criteria

- [ ] `WiFiUDP` socket opened once in `setup()` targeting `XR18_IP:XR18_PORT`
- [ ] `loop()` checks `g_bpm_changed`; when true, clears flag and calls `send_bpm_osc(bpm)`
- [ ] `send_bpm_osc()` builds a valid OSC `/fx/{slot}/par/02 ,f {val}` packet
- [ ] OSC packet byte layout verified against `src/X32lib/Xsprint.c` reference
- [ ] XR18 FX slot delay time visibly changes in X-Air Edit when DAW BPM changes
- [ ] BPM range tested: 60, 120, 180 BPM → correct normalised values (see Notes)

## Notes

**Firmware scope**: `esp32/X32Link/` only. Do NOT touch `esp32/X32_emulator/`.

**BPM → normalised float** (inline — cannot share `src/x32tap_logic.c`):
```c
// Mirrors src/x32tap_logic.c: ms_to_normalized(), TAP_MAX_MS = 3000
static float bpm_to_normalized(float bpm) {
    if (bpm <= 0.0f) return 0.0f;
    float ms = 60000.0f / bpm;
    if (ms > 3000.0f) ms = 3000.0f;
    return ms / 3000.0f;
}
```

**Expected values**:
| BPM  | delay_ms | normalised |
|------|----------|------------|
|  60  | 1000 ms  | 0.3333     |
| 120  |  500 ms  | 0.1667     |
| 180  |  333 ms  | 0.1111     |
| 200  |  300 ms  | 0.1000     |

**OSC packet format** for `/fx/1/par/02 ,f 0.1667`:
```
2f 66 78 2f  31 2f 70 61  72 2f 30 32  00 00 00 00   /fx/1/par/02....
2c 66 00 00                                           ,f..
3e 2a a3 d7                                           (float 0.1667 big-endian)
```
Total: 20 bytes. OSC strings are null-padded to 4-byte boundaries.
Float is IEEE 754 big-endian (network byte order).

**OSC build helper**:
```cpp
static int osc_str(uint8_t *buf, int idx, const char *s) {
    int len = strlen(s) + 1;           // include null terminator
    memcpy(buf + idx, s, len);
    idx += len;
    while (idx % 4) buf[idx++] = 0;   // pad to 4-byte boundary
    return idx;
}

void send_bpm_osc(float bpm) {
    char path[20];
    snprintf(path, sizeof(path), "/fx/%d/par/02", LINK_FX_SLOT);
    float val   = bpm_to_normalized(bpm);
    uint32_t fbe;                      // float big-endian
    memcpy(&fbe, &val, 4);
    fbe = htonl(fbe);

    uint8_t pkt[32] = {0};
    int idx = osc_str(pkt, 0, path);
    idx     = osc_str(pkt, idx, ",f");
    memcpy(pkt + idx, &fbe, 4); idx += 4;

    udp.beginPacket(XR18_IP, XR18_PORT);
    udp.write(pkt, idx);
    udp.endPacket();
}
```

**OSC reference**: `docs/xr18-geq-osc.md` — `/fx/{slot}/par/02` is the
delay time parameter (same address that `src/X32Tap.c:do_tap()` uses at line 101).

**XR18 port**: 10024 (XAir/XR18). Not 10023 (that is X32).
