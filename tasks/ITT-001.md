## Goal

Create the `esp32/MidiOscIttt/` Arduino sketch — an ESP32-S3 "If This Then That"
bridge between **USB-MIDI** and **mixer OSC**, in both directions. MIDI events
trigger OSC commands and observed OSC changes trigger MIDI out, driven by a
user-configurable rules table (engine lands in ITT-002). Works on X32 (10023)
and XR18 (10024).

This task is scaffold + both I/O legs wired + pass-through logging.

## Notes

- Reuse the MIDI infrastructure already host-tested in `X32MidiClock/`
  (`midi_*`); this firmware consumes Note/CC/PC, not just clock.
- No rule evaluation in this task — just prove both I/O legs deliver events to a
  single dispatch point.
- MIDI-before-WiFi enumeration is a hard ordering requirement on ESP32-S3
  (`tempo_source_pre_net()` precedent).
