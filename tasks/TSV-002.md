## Goal

Implement `esp32/X32ToastSaver/mic_capture.cpp` and `mic_capture.h`: initialize the INMP441 MEMS microphone over I2S DMA and deliver 1024-sample frames (int32_t, L-channel only) to the caller on demand.

## Hardware: INMP441

The INMP441 is a digital I2S MEMS microphone:
- Output: 24-bit signed PCM in 32-bit I2S frames, MSB-justified
- L/R select: tie the L/R pin to GND for left channel (word select low = left)
- Max sample rate: 62.5 kHz; use 48 kHz
- Pins required: BCLK (clock), WS (word select / LRCK), DATA (serial data out from mic)

Wiring to ESP32-S3 (pins from `config.h`):
```
INMP441  →  ESP32-S3
BCLK     →  TSV_I2S_BCLK   (GPIO 4)
WS       →  TSV_I2S_LRCLK  (GPIO 5)
SD       →  TSV_I2S_DIN    (GPIO 6)
L/R      →  GND
VDD      →  3.3V
GND      →  GND
```

## Public interface (mic_capture.h)

```c
#pragma once
#include <stdint.h>
#include <stdbool.h>

/* Initialize I2S peripheral and DMA buffers. Call once from setup(). */
bool mic_capture_begin(void);

/* Fill buf[TSV_FFT_SIZE] with the next frame of int32_t PCM samples.
 * Blocks until TSV_FFT_SIZE samples are available (DMA transfer complete).
 * Returns false on I2S read error. */
bool mic_capture_read(int32_t *buf);

/* Stop I2S and free DMA buffers. */
void mic_capture_end(void);
```

## I2S configuration (ESP-IDF v5 / Arduino ESP32 core 3.x)

```cpp
#include <driver/i2s_std.h>

static i2s_chan_handle_t s_rx_chan = NULL;

bool mic_capture_begin(void) {
    i2s_chan_config_t chan_cfg = I2S_CHANNEL_DEFAULT_CONFIG(
        (i2s_port_t)TSV_I2S_PORT, I2S_ROLE_MASTER);
    chan_cfg.dma_desc_num  = 4;
    chan_cfg.dma_frame_num = TSV_FFT_SIZE;   /* 1024 samples per DMA buffer */

    if (i2s_new_channel(&chan_cfg, NULL, &s_rx_chan) != ESP_OK) return false;

    i2s_std_config_t std_cfg = {
        .clk_cfg  = I2S_STD_CLK_DEFAULT_CONFIG(TSV_SAMPLE_RATE_HZ),
        .slot_cfg = I2S_STD_MSB_SLOT_DEFAULT_CONFIG(I2S_DATA_BIT_WIDTH_32BIT,
                                                     I2S_SLOT_MODE_MONO),
        .gpio_cfg = {
            .mclk = I2S_GPIO_UNUSED,
            .bclk = (gpio_num_t)TSV_I2S_BCLK,
            .ws   = (gpio_num_t)TSV_I2S_LRCLK,
            .dout = I2S_GPIO_UNUSED,
            .din  = (gpio_num_t)TSV_I2S_DIN,
            .invert_flags = { .mclk_inv=false, .bclk_inv=false, .ws_inv=false },
        },
    };
    if (i2s_channel_init_std_mode(s_rx_chan, &std_cfg) != ESP_OK) return false;
    return i2s_channel_enable(s_rx_chan) == ESP_OK;
}

bool mic_capture_read(int32_t *buf) {
    size_t bytes_read = 0;
    esp_err_t err = i2s_channel_read(s_rx_chan, buf,
                                     TSV_FFT_SIZE * sizeof(int32_t),
                                     &bytes_read, portMAX_DELAY);
    return err == ESP_OK && bytes_read == TSV_FFT_SIZE * sizeof(int32_t);
}

void mic_capture_end(void) {
    if (s_rx_chan) {
        i2s_channel_disable(s_rx_chan);
        i2s_del_channel(s_rx_chan);
        s_rx_chan = NULL;
    }
}
```

## Acceptance Criteria

- [ ] `mic_capture.h` defines the three-function interface above.
- [ ] `mic_capture.cpp` implements all three using `driver/i2s_std.h` (ESP-IDF v5 API — not the deprecated `driver/i2s.h`).
- [ ] `mic_capture_begin()` is called from `setup()` after WiFi connects; prints `[TSV] mic init OK` or `[TSV] mic init FAILED` to Serial.
- [ ] `loop()` calls `mic_capture_read(frame)` and prints the peak int32 value of the frame every 50 frames (approx once per second), confirming mic is alive.
- [ ] Sketch still compiles and the peak value changes when sound is present near the mic.

## Notes

- The INMP441 outputs 24-bit data in a 32-bit word, left-justified (bit 31 is MSB of the audio sample). Right-shift by 8 before use: `int32_t sample = raw >> 8`.
- `I2S_STD_MSB_SLOT_DEFAULT_CONFIG` with `I2S_SLOT_MODE_MONO` reads only the left channel (WS-low slot), which is what the INMP441 produces when L/R is tied to GND.
- DMA frame count of 1024 means `mic_capture_read()` delivers exactly `TSV_FFT_SIZE` samples in one call, matching what `fft_detector` needs.
- If the board's I2S pins conflict with the LCD, check the Waveshare ESP32-S3-Touch-LCD-4.3 schematic and update the `TSV_I2S_*` pin defines in `config.h` accordingly.
- The Arduino ESP32 core 3.x includes ESP-IDF v5.x; use `#include <driver/i2s_std.h>`, not the legacy `<driver/i2s.h>`.
